Module endhook

Run a hook after the main script is read. Requires the debug library.

Functions

hook (f, level) Set the hook.


Functions

hook (f, level)
Set the hook.

Parameters

  • f: The function to call.
  • level: How many stack levels away from this call to hook. Defaults to 4, so a file requiring a library calling this is hooked.
